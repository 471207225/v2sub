# v2sub
v2ray linux订阅切换脚本

首先，确认你的系统已经安装了python3,centos7由于库比较老，可能需要自行编译安装
食用方法：

``````
git clone https://github.com/xbblog95/v2sub.git;
cd v2sub;
chmod 777 v2sub;
./v2sub;
``````
